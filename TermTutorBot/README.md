# Telegram-бот "Term Tutor"
**TermTutorBot** - це розумний Telegram-бот, створений на базі `C# (.NET)` та `OpenAI API`. Його головна мета - швидко і зрозуміло пояснювати невідомі слова чи терміни, а також надавати приклади їх використання, синоніми та цікаві факти через зручне меню інлайн-кнопок.
--
# Що вміє цей бот?
Ви надсилаєте незрозуміле слово чи термін, бот пише його значення і пропонує варіанти:
- Речення із цим словом
- Синоніми до цього слова, якщо такі є
- Додаткова інформаціія про слово

# Як запустити цього Telegram-бота?
Для запуску бота вам потрібно:
- у застосунку Telegram зайти у [BotFather](https://t.me/BotFather) та створити бота за інструкцією. Збережіть отриманий токен
- встановіть останню версію [.NET](https://dotnet.microsoft.com/en-us/download), якщо цього не було зроблено раніше
- також у вас має бути ключ OpenAI API<br/>

Далі створіть директорій на своєму комп'ютера. Відкрийnt цю директорію у зрeчному редакторі. Скопіюйте цей репозиторій та перейдіть у директорію `TermTutorBot`:
```
https://github.com/Arturich666/Term-Tutor-Telegram-Bot.git
```
У папці `TermTutorBot` створіть файл `appsettings.json`, скопіюйте код нижче:
```json
{
     "BotSettings": {
       "TelegramToken": "ТВІЙ_TELEGRAM_BOT_TOKEN",
       "OpenAIApiKey": "ТВІЙ_OPENAI_API_KEY",
       "Model": "gpt-4o-mini",
       "SystemPrompt": ""
     }
}
```
Для запуску відкрийте термінал та виконайте команду:
```
dotnet run
```
Якщо все налаштовано правильно, у консолі з'явиться повідомлення: Бот @НазваБота запущений.
# Технологічний стек:
- C# / .NET 8
- Telegram.Bot - клієнт для Telegram Bot API
- OpenAI API (/v1/responses)
- HttpClient - для роботи з API
- System.Text.Json - обробка JSON
